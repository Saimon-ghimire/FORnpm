A computer is an electronic devices that takes in input, processes it and gives meaningful result. Computer is made up of hardware and software components. Hardware components of a computer can be classified as:

1. Input devices:

The devices used to feed in raw data and instructions to the computer. For eg: keyboard, mouse, microphone, etc.

2. Processing devices:

The devices that process the raw data given by the user according to the instructions and produces meaningful output. Processing devices can be classified as ALU, CU and Registers.

3. Output devices:

The devices that display the processed result to the user. For eg: monitor, speaker, projecter, etc.
